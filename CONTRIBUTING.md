# Contributing

Thank you for your interest in contributing.

## How to Contribute

- **Found a bug or have a suggestion?** → [Open an issue](https://github.com/hyperfinitism/workflow-recipes/issues).
- **Want to fix a bug, add a recipe, or refactor something?** → [Open a pull request](https://github.com/hyperfinitism/workflow-recipes/pulls).

> [!WARNING]
> Do not report a vulnerability-related issue or submit a patch directly. If you find a vulnerability, please use [Private vulnerability reporting](https://github.com/hyperfinitism/workflow-recipes/security).
